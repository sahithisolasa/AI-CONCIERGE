# AI CONCIERGE

## Overview
AI CONCIERGE is a full-stack AI-powered financial assistant that helps users discover Economic Times services through personalized recommendations.

## Features
- User profiling
- Recommendation engine
- Clean UI dashboard
- FastAPI backend

## Tech Stack
- Frontend: Streamlit
- Backend: FastAPI
- Language: Python

## Run

Backend:
uvicorn backend.main:app --reload

Frontend:
streamlit run frontend/app.py
